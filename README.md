# Уездный кондитер 🍰

Интернет-магазин кондитерской с конструктором тортов

## 🚀 Технологии

- **Frontend:** Next.js 14, TypeScript, SCSS
- **Backend:** Supabase (PostgreSQL, Auth, Storage)
- **Анимации:** GSAP
- **Платежи:** YooKassa
- **Деплой:** Vercel

## 📋 Возможности

### Для покупателей
- 🛒 Каталог товаров с фильтрами
- 🎂 Интерактивный конструктор тортов
- 📦 Корзина и оформление заказа
- 💳 Онлайн оплата
- 👤 Личный кабинет
- ⭐ Система отзывов
- 🎁 Бонусная программа

### Для администраторов
- 📊 Управление товарами и заказами
- 🏷️ Акции и промокоды
- 📝 Управление контентом (блог, страницы)
- 📈 Аналитика продаж
- 🔔 Уведомления
